System Monitoring Program

This program provides a simple command-line interface for monitoring system information and managing system resources on Linux systems. It includes options to display various system parameters, add system snapshots to history, execute a system resource monitor script, and more.
Files Included:

    Peoject 3.c: The main C code for the system monitoring program.
    Project 3.sh: A Bash script for monitoring system resources (CPU, memory, disk usage).

Usage:

Follow the usage guide provided above to compile and run the system monitoring program on your Linux system. Ensure that you have appropriate permissions to execute Bash scripts and compile C code.
